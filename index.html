Proyecto 1


WeforzTech

WeforzTech es una tienda online (no real) de venta de hardware y software. En ella podemos hacer compras de articulos tanto hardware como software, o hacerlo a medida, con un coste extra. Podemos ver proyectos ya realizados. Cuenta con un extenso panel de administración

  • javascript

Proyecto 2


Pedrolos-T2

Pedrolos T2 es una web para un servidor de Minecraft. En ella podemos ver el estado de este, actualizaciones, noticias, un mapa dinámico que cambia cada semana, jugadores que juegan y una galería de fotos. Cuenta con un sistema de inicio de sesión para los administradores


  • html5
  • css3
  • php
  • javascript
  • mysql

Tesla Web Clon (Próximamente)

Clon prácticamente exacto de la WEB de Tesla España. Se ha simulado tanto la página principal, como los configuradores de cada vehículo; así como los productos de carga y paneles solares.
